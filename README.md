# changelog-slugs

`changelog-slugs` provides changelog-aware heading slugs for
[Python-Markdown](https://python-markdown.github.io/)’s
[`toc` extension](https://python-markdown.github.io/extensions/toc/).

It is designed for [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
style changelogs, where repeated subsection headings like `Added` and `Fixed`
would otherwise receive unstable, order-dependent autogenerated anchors. As
new releases are added above older ones, those anchors can change and break
existing links. `changelog-slugs` instead scopes subsection slugs to their
containing release, producing stable anchors such as `v1-2-3-fixed`.
